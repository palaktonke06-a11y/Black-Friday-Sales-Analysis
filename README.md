# 📌 Project Overview

The Black Friday Sales Analysis project explores customer purchasing behavior during Black Friday sales using Exploratory Data Analysis (EDA) and Feature Engineering techniques. The dataset was cleaned, transformed, and analyzed to identify spending patterns and key factors influencing customer purchases.

Using Python, Pandas, NumPy, Matplotlib, and Seaborn, the project provides valuable insights into customer preferences and purchasing trends that can support data-driven business decisions.
---

# 🎯 Project Objectives

* Understand customer purchasing behavior.
* Analyze factors affecting purchase amounts.
* Perform data cleaning and preprocessing.
* Apply feature engineering techniques.
* Visualize customer trends and patterns.
* Generate actionable business insights.

## Key EDA Insights

* Male customers generally contribute higher purchase amounts compared to female customers.
* Customers belonging to the 26–35 age group represent the highest spending segment.
* Purchasing behavior varies significantly across occupations.
* Product categories have a strong influence on customer spending patterns.
* Different customer groups exhibit distinct purchasing preferences.

# ⚙️ Feature Engineering

Feature Engineering was performed to prepare the dataset for analysis and future machine learning applications.

## Feature Engineering Tasks Performed

| Task                                  | Description                                           |
| ------------------------------------- | ----------------------------------------------------- |
| Missing Value Handling                | Identified and handled missing values                 |
| Categorical Data Encoding             | Converted object-type variables into numerical values |
| Gender Transformation                 | Transformed gender categories into numerical format   |
| Age Group Encoding                    | Converted age categories into machine-readable values |
| City Category Transformation          | Encoded city categories into numerical representation |
| Stay in Current City Years Processing | Converted categorical values into integer format      |
| Product Category Processing           | Handled category-based transformations                |
| Data Type Conversion                  | Converted object columns into integer values          |
